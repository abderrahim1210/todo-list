# To-Do List 📝

A simple and elegant To-Do List web application built with HTML, CSS, JavaScript, and Bootstrap.

## 🔧 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive UI with Bootstrap styling

## 🛠 Technologies Used
- HTML5
- CSS3 (custom styles + Bootstrap)
- JavaScript (Vanilla JS)
- Bootstrap 5

## 🚀 How to Use
1. Download or clone this repository.
2. Open `todo-list.html` in your browser.
3. Start adding your tasks and managing your to-do list.

## 📁 Project Structure
- `todo-list.html` – Main HTML file  
- `todo-list.css` – Custom styles for the list  
- `todo-list.js` – JavaScript for functionality  
- `bootstrap.min.css` – Bootstrap styles  
- `bootstrap.bundle.min.js` – Bootstrap JS components  
- `task icon.png` – Project icon  

## 📌 Notes
- This project is fully client-side and does not require a server or database.
- All tasks are stored temporarily and will reset on page refresh.
